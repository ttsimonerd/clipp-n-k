# clipp'n'k

### **Own your clips. Own your platform.**

A **free**, **open-source**, and **self-hostable** clip sharing platform
built around Discord.

Use the **Official Hosted Instance** for free or deploy your own
instance for complete control.

[![Website](https://img.shields.io/badge/Website-clipp--n--k-E8401A?style=for-the-badge)](https://clipp-n-k.duckdns.org)
[![Discord](https://img.shields.io/discord/1529491880725840083?style=for-the-badge&logo=discord&label=Discord)](https://discord.gg/e3sHxNmGMf)
[![License](https://img.shields.io/github/license/ttsimonerd/clipp-n-k?style=for-the-badge)](LICENSE)

> **Your Discord community deserves its own home for clips.**

clipp'n'k is built around one simple idea: **your content should belong
to your community---not a third-party platform.**

Use the **Official Hosted Instance** to start sharing instantly, or
self-host your own deployment to control every aspect of your storage,
branding, limits and infrastructure.

# ✨ Why clipp'n'k?

-   🆓 Free to use
-   🌍 Open Source
-   🏠 Self-Hostable
-   🎮 Discord-native authentication
-   📦 Automatic compression (up to **80%**)
-   🔒 Encrypted files & database
-   ☁️ Local or S3-compatible storage
-   🔗 Shareable public links
-   👥 Community-first

------------------------------------------------------------------------

# 🚀 Features

  -----------------------------------------------------------------------
  Feature                       Description
  ----------------------------- -----------------------------------------
  🎥 Upload & Share             Upload clips and share them using simple
                                links.

  📦 Smart Compression          Reduce file sizes by **up to 80%**. A **1
                                GB** upload can often become **\~200 MB**
                                while preserving quality.

  🔒 Secure Storage             Uploaded files and database records are
                                encrypted.

  🎮 Built Around Discord       Authenticate with Discord and easily
                                share clips within your communities.

  🌎 Official Hosted Instance   Free to use with uploads up to **1 GB**.

  🏠 Self-Hosting               Deploy with Docker and configure storage,
                                branding, upload limits and more.
  -----------------------------------------------------------------------

------------------------------------------------------------------------

# 🌍 Official Hosted vs Self-Hosted

  Official Hosted Instance    Self-Hosted
  --------------------------- -------------------------------
  ✅ Ready to use             ✅ Full control
  ✅ Free                     ✅ Your infrastructure
  ✅ Uploads up to **1 GB**   ✅ Configurable upload limits
  ✅ Automatic updates        ✅ Custom branding
  ✅ No setup                 ✅ Local or S3 storage
  Great for individuals       Perfect for communities
